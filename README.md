# PTS 业务网数据服务程序
## 1. 程序概述
PTS 业务网数据服务程序，用于业务网数据采集，进行数据解析，并将解析后的数据存储到数据库中。

## 2. 程序结构
程序主要由以下几个部分组成：
- `main.py`：程序入口，负责启动服务，监听端口，接收数据，调用数据处理函数。
- `log/PTS_Traffic.log`：数据日志文件。
- `file/PTS_edit.xlsx`：数据表格文件。

## 3. 程序功能
程序的主要功能包括：
- 监听端口，接收数据。
- 解析数据，提取关键信息。
- 将解析后的数据存储到数据库中。

## 4. 程序使用
程序的使用方法如下：
1. 确保已经安装了Python 3.6及以上版本。
2. 在命令行中进入程序所在的目录。
3. 运行命令`python main.py`启动程序。
4. 程序会监听端口，等待数据接收。

## 5. 使用目录
要创建以下目录才能使用脚本:
```
mkdir ./traffic/{acc_line,ce_device_interface,ce_main_interface,logical_node_line,pe_physical_interface,physical_node_line}
```

## 6. 注意事项
- 请确保在使用程序之前已经正确配置了数据库连接信息。
- 请确保在使用程序之前已经正确配置了数据解析规则。
- 请确保在使用程序之前已经正确配置了数据存储规则。
- 请确保在使用程序之前已经正确配置了日志文件路径。
- 请确保在使用程序之前已经正确配置了数据表格文件路径。
